  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with AQI Predictor in minutes.</p>
  
  <Steps>
    <Step title="Install the package">
      Install AQI Predictor using pip:
      
      ```bash
      pip install aqi-predictor
      ```
    </Step>
    
    <Step title="Configure your API key">
      Set up authentication by configuring your API credentials:
      
      ```python
      from aqi_predictor import AQIClient

      client = AQIClient(api_key="your_api_key_here")
      ```
      
      <Note>Get your API key from the dashboard after creating an account.</Note>
    </Step>
    
    <Step title="Make your first prediction">
      Submit environmental data to get an AQI prediction:
      
      ```python
      prediction = client.predict({
          "temperature": 25.5,
          "humidity": 65,
          "pm25": 12.3,
          "pm10": 22.1,
          "no2": 15.2,
          "location": {"lat": 37.7749, "lon": -122.4194}
      })

      print(f"Predicted AQI: {prediction.aqi}")
      print(f"Category: {prediction.category}")
      ```
      
      <Accordion title="Example Response">
        ```json
        {
          "aqi": 42,
          "category": "Good",
          "confidence": 0.94,
          "timestamp": "2024-03-15T10:00:00Z",
          "components": {
            "pm25": 12.3,
            "pm10": 22.1,
            "no2": 15.2
          }
        }
        ```
      </Accordion>
    </Step>
  </Steps>
